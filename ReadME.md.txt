# TaskMaster-Java 🗂️

A simple and intuitive Java-based Task Management System with a graphical user interface (GUI). Built as a second-year college project to help users manage their daily tasks efficiently, with options to save tasks to a file or a database.

---

## 🔧 Features

- 🖥️ User-friendly GUI built with Java Swing
- ➕ Add, edit, and delete tasks
- 💾 Option to save tasks to a file or a database
- 🔍 Task filtering and sorting (if applicable)
- 🧠 Designed for productivity and simplicity
